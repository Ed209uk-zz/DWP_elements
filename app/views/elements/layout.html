DWP elements Layout

Use the standard GOV.UK layout guidelines wherever possible.

Consider whether you can design a better experience by using the full width of the screen eg for a staff-facing service that will only be used on a wide-screen monitor. Use the menu pattern in the DWP pattern library for sub-navigation in a staff-facing service.
